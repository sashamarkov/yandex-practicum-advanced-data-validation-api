Вам необходимо реализовать генератор текста-рыбы Lorem Ipsum, такой текст будет полезен для дизайнеров и для тестировщиков. Ваш сервер уже умеет генерировать текст с помощью специальной библиотеки.
Запросите по ссылке https://webinars.webdev.education-services.ru/train/7/2/4/lorem с помощью GET-запроса и выведите его на страницу.

---
Не забывайте про await, функции fetch() асинхронные
Сервер генерирует произвольный текст при каждом запросе.
Для этой задачи достаточно текстового формата.
