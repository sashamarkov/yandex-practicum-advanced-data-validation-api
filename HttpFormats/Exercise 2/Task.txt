Цитаты исполнителя Kanye West набирают популярность, поэтому вам уже нужен отдельный сервер, чтобы распространять их. Такой промежуточный сервер называется "прокси".
Запросите цитату через прокси-сервер https://webinars.webdev.education-services.ru/train/7/2/5/kanye, затем обработайте цитату и выведите её на экран. Обратите внимание, промежуточный сервер может немного поменять формат данных.

---
Функция fetch.json() разбирает JSON из тела ответа
Функция fetch.json() не может разобрать JSON из внутренних полей в JSON-теле ответа.
Функция JSON.parse() разбирает JSON из любого текста
