Вам необходимо создать сервис регистрации пользователей. Они должны заполнить форму с ФИО, а вы затем отправить её с помощью fetch() на специальный ресурс проверки регистрации.
Две особенности: вам необходимо использовать GET-запрос и затем нужно использовать промис, который вернёт fetch() (то есть в этом тренажёре нужно обойтись без async/await).
Сконвертируйте полученные данные формы в URLSearchParams и соедините с основной ссылкой.

---
Метод POST не подойдёт для этого задания
fetch() возвращает промис, а также .json() возвращает промис, воспользуйтесь этим
Сервер вернёт JSON с теми данными формы, которые вы отправили
