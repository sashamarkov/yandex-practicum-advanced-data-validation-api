Методом fetch сделайте простой запрос по адресу:
https://swapi.nomoreparties.co/people 
В обработчике then выведите ответ в консоль. Выведите отдельно код статуса запроса и сам статус, затем полученный результат целиком. С помощью функции .text() выведите ответ сервера в виде текста.

---
Методу fetch нужно передать один аргумент — строку
Чтобы открыть DevTools для тестов, нажмите Ctrl+Shift+J
Функция res.text() асинхронная

---
index.js

...'https://swapi.nomoreparties.co/people'... //используйте fetch
  .then(res => {
    console.log(...) // выведите код и сообщение статуса
    console.log(res) // выведем результат, можно изучить объект, хотя в нём почти ничего нет
    ...  //запросите и выведите текст
      .then(text => console.log(text))
  });
